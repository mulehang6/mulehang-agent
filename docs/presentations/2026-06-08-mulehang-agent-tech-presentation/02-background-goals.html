项目目标与当前范围

本次展示聚焦当前已经完成的能力：Koog 接入、内置文件工具接入、工作区内的简单代码探索，以及 CLI 流式输出。

项目目标

  • 基于 JetBrains Koog 搭建一个可以运行的 agent 原型。
  • 让 agent 能调用内置文件工具，在工作区内完成目录查看和文件读取。
  • 让运行结果通过 runtime 流式返回到 CLI 界面。

当前完成度集中在 代码库探索能力，展示重点也是这条链路。

当前展示范围

  • 讲 Koog agent 如何注册工具。
  • 讲 built-in file tools 如何受工作区根目录限制。
  • 讲 CLI 如何接收 `tool.call`、`text.delta` 这类流式事件。
