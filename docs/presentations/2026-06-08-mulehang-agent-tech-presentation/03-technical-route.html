Scope

当前实现范围

当前实现范围收敛在一条清晰链路：请求进入 runtime，Koog 调用文件工具，对工作区进行简单探索，再把结果流式返回到 CLI。

01

Runtime Request

CLI 通过 `/runtime/run/stream` 发起一次运行请求。

02

Koog Agent

RuntimeAgentExecutor 调用组装后的 Koog agent 执行 prompt。

03

Built-in Tools

KoogToolRegistryAssembler 注册目录查看和文件读写工具。

04

Workspace Guard

WorkspaceScopedFileSystemProvider 限制工具只能访问工作区路径。

05

Tool Call Event

运行中会产生 `tool.call` 和 `text.delta` 等事件。

06

CLI Display

RuntimeHttpClient 解析 SSE，并映射为 UI 消息。

07

Simple Exploration

最终支持查看目录、读取文件、做简单代码库探索。

当前重点 1

把 Koog 内置文件工具安全接进 runtime。

当前重点 2

把工具执行过程通过流式事件暴露给 CLI。

当前重点 3

让 agent 可以完成简单的代码库探索任务。
