Built-in File Tools

核心实现：Koog 内置文件工具接入

当前最实的一块能力是文件工具接入。runtime 会默认挂载 4 个内置文件工具，本次展示重点放在目录查看和文件读取这两个能力。

工具集合

  • `__list_directory__`
  • `__read_file__`
  • `__write_file__`
  • `edit_file`

课堂展示主要会用到 `__list_directory__` 和 `__read_file__`。

默认启用位置

  • `defaultRuntimeCapabilitySet()` 会把这 4 个工具放进默认 capability set。
  • 对应实现位于 `RuntimeHttpService.kt`。

注册位置

  • `KoogToolRegistryAssembler` 会把这些 built-in file tools 注册进 primary registry。
  • 测试已经验证工具名和风险级别会进入 runtime 执行链。
