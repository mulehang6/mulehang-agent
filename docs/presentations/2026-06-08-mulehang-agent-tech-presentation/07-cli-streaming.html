CLI Surface

核心实现三：CLI 流式输出

CLI 当前通过 `runtime-http-client.ts` 连接 `/runtime/run/stream`，把 SSE 事件映射成界面消息。这里已经能看到工具调用和文本增量输出。

CLI 当前职责

  • 接收用户输入与命令
  • 连接 runtime HTTP client
  • 消费 SSE 事件流并更新界面
  • 显示 `tool.call`、`text.delta`、`status` 等事件

当前比较关键的前端文件

  • `runtime-http-client.ts`
  • `runtime-events.ts`
  • `components/TranscriptView.tsx`
  • `screens/ChatScreen.tsx`
